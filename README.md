# Трекер прочитанных книг

Консольное приложение на Python для отслеживания прочитанных книг с сохранением данных в JSON.

## Возможности

- Добавление книг с проверкой на дубликаты
- Просмотр всех добавленных книг
- Расчёт средней оценки по всем книгам
- Статистика количества книг по авторам
- Удаление книг (по индексу или по автору+названию)
- Автосохранение в JSON файл

## Запуск

```bash
python main.py
